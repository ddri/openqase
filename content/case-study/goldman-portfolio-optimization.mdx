# Portfolio Optimization at Goldman Sachs

## Overview

Goldman Sachs implemented quantum algorithms to optimize their investment portfolio allocation, achieving improved risk-adjusted returns compared to traditional methods. The implementation utilized the Quantum Approximate Optimization Algorithm (QAOA) to address complex portfolio optimization challenges.

## Implementation Details

### Technical Architecture
- IBM Quantum Hardware for computation
- Qiskit Runtime for execution
- QAOA implementation for optimization
- Custom tools for portfolio management

### Key Metrics
- Sharpe Ratio Improvement: 20%
- Computation Time Reduction: 50%
